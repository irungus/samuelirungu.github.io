Geospatial AI Project

🌳 Tree Species Classification
using Geospatial ML

An end-to-end pipeline using multi-source remote sensing data (Sentinel-1 & 2)
to classify forest species with high precision.

📌 Overview

This project implements an end-to-end geospatial machine learning pipeline for classifying tree species using multi-source remote sensing data. By integrating optical and radar data, the model overcomes cloud cover limitations typical in Kenyan forest regions.

🚀 Key Features

  • Feature extraction from Sentinel-1 (SAR) and Sentinel-2 (optical).
  • Vegetation indices calculation (NDVI, EVI, SAVI).
  • Terrain and climate data integration.
  • Advanced ML: Random Forest, XGBoost, and LightGBM.

⚙️ Workflow & Architecture

The project follows a rigorous 5-step geospatial workflow:

  1. Data Extraction: Using Google Earth Engine (GEE) to pull multi-spectral bands.
  2. Feature Engineering: Preprocessing terrain and climate variables.
  3. Model Training: Comparative analysis between XGBoost and LightGBM.
  4. Evaluation: Hyperparameter tuning via Grid Search.
  5. Deployment: Mapping predicted species over the target region.

📊 Performance Results

Best Model: XGBoost
Accuracy: 75%
Key Finding: Integrating Sentinel-1 SAR data improved classification accuracy by 12% in high-canopy areas.
